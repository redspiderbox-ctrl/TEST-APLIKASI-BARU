Panduan Setup Firebase

  1. Buka Firebase Console dan buat project baru (atau pilih yang sudah ada).
  2. Masuk ke menu Build → Realtime Database, klik Create Database.
  3. Pilih region terdekat, lalu pilih Start in test mode (untuk development).
  4. Klik ikon (Project Settings) di sidebar kiri atas.
  5. Scroll ke bawah, klik Add app → Web app (ikon </>).
  6. Daftarkan nama app, lalu copy konfigurasi firebaseConfig yang muncul.
  7. Paste masing-masing nilai ke form di bawah ini, lalu klik Hubungkan.
