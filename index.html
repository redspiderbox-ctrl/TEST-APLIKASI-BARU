Konfigurasi Firebase

Masukkan konfigurasi Firebase Realtime Database Anda. Data ini disimpan di localStorage browser.

Panduan Setup Firebase
  1. Buka Firebase Console
  2. Buat project baru atau pilih project yang sudah ada
  3. Buka Build → Realtime Database → Create Database
  4. Pilih region terdekat, lalu pilih Start in test mode
  5. Buka Project Settings → General → Your apps → Web app
  6. Salin konfigurasi dan paste di atas
